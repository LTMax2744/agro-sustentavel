# Agro Forte, Futuro Sustentável

Site informativo sobre agronegócio sustentável desenvolvido para o Concurso Agrinho 2026 (Subcategoria 3 - Programação Front-End).

## Sobre o Projeto

Este site apresenta o tema **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"** através de uma interface moderna e profissional.

**Tecnologias utilizadas:**
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (React 19)
- Tipografia: Playfair Display (títulos) e Inter (corpo)

## Seções do Site

1. **Navegação** - Menu sticky com links para as principais seções
2. **Hero Section** - Apresentação do tema com chamada para ação
3. **Tema Central** - Explicação do equilíbrio necessário com estatísticas
4. **Práticas Sustentáveis** - 6 cards com práticas principais (Rotação de Culturas, Irrigação Eficiente, Manejo Integrado, Agricultura de Precisão, Energia Renovável, Certificação Ambiental)
5. **Impacto** - Dados visuais sobre resultados mensuráveis
6. **Futuro** - Visão de futuro com metas para 2030
7. **Footer** - Links e informações adicionais

## Design

**Paleta de cores:**
- Verde floresta: #2D5016 (primário)
- Marrom terra: #8B7355 (secundário)
- Verde claro: #A4D65E (acentos)
- Branco: #FFFFFF (fundo)

**Características:**
- Design limpo e minimalista
- Animações suaves ao scroll
- Responsivo (mobile, tablet, desktop)
- Navegação intuitiva
- Cards com efeitos hover

## Como Usar

1. Clone ou baixe o repositório
2. Instale as dependências: `pnpm install`
3. Inicie o servidor de desenvolvimento: `pnpm dev`
4. Abra http://localhost:3000 no navegador

## Deployment

Para fazer deploy:

1. Build: `pnpm build`
2. Publique a pasta `dist` no GitHub Pages, Vercel ou outro serviço de hosting
